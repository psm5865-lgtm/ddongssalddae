# 똥쌀때

> 변기 위 5분 마음챙김 — 숏츠 대신 단 하나의 활동에만 머물러요. (토스 미니앱 컨셉)

화장실에서 핸드폰을 놓지 못하는 습관을, 무한 스크롤 대신 **딱 하나의 짧고 회복적인 활동**(호흡·감사·목표·명언 중 랜덤 1개)으로 바꿔주는 앱이에요. 선택의 함정·피드·추천 콘텐츠를 구조적으로 제거했고, 끝나면 조용히 도장 하나만 남겨요.

## 개발

```bash
npm install
npm run dev      # 개발 서버
```

## 빌드

```bash
npm run build    # 타입체크 + dist/ 정적 SPA 생성
npm run preview  # 빌드 결과 로컬 확인
```

## 기술 스택

- Vite + React 18 + TypeScript
- React Router (4 화면: 온보딩 / 홈 / 세션 / 완료 + 설정)
- framer-motion (도장·전환 모션)
- localStorage (서버·계정 없음 — 기록·설정 전부 기기 안에)
- `@vercel/analytics` (쿠키리스, 배포 시 대시보드에서 켜기)

## 핵심 원칙 (비협상)

1. 활동은 매번 시스템이 랜덤으로 1개 정함 — 선택 화면 없음
2. 앱 안에 피드·둘러보기·외부 링크 없음
3. 알림은 **기본 OFF** — 앱이 먼저 연락하지 않음
4. 기록은 조용한 도장만 — 점수·연속 배지·강제 푸시 없음
